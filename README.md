# Python Assignment Part 3 — Product Explorer & Error-Resilient Logger

## Description
This project demonstrates file handling, API integration, exception handling, input validation, and logging in Python. The program reads and writes files, fetches product data from an API, handles errors safely, and logs errors into a file.

## Tasks Implemented

### Task 1 — File Read & Write
- Write student notes to python_notes.txt
- Append additional lines
- Read file and print numbered lines
- Search keyword in file

### Task 2 — API Integration
- Fetch products from DummyJSON API
- Display product table
- Filter products by rating
- Sort by price
- Fetch laptops category
- Simulate POST request

### Task 3 — Exception Handling
- Safe divide function
- Guarded file reader
- Input validation loop
- API product lookup

### Task 4 — Logging to File
- Log errors into error_log.txt
- Log connection error
- Log HTTP 404 error
- Print log file contents

## Files in Repository
- part3_api_files.py
- python_notes.txt
- error_log.txt
- README.md

## How to Run
1. Install requests library:
   pip install requests

2. Run the program:
   python part3_api_files.py
